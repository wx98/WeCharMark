# WeCharMark

这是一个基于WebService的课程与成绩管理系统   
服务端使用三层架构并在IIS上发布   
客户端使用WPF编写窗口
## 实现功能
  1. **基本的登录退出**操作
  1. **学生的增删改查**操作
  2. **课程的增删改查**操作
  3. **成绩的增删改查**操作
  4. **成绩的批量导入**操作

## 项目编译运行环境
* **平台:** windows 7(或者以上版本)
* **程序:** Microsoft Visual Studio Ultimate 2012
* **框架:**.NET Framework 4.5
* **数据库：** Microsoft SQL Server
* **服务端:** Internet Information Services (IIS)管理器
